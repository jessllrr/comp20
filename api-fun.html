Boost of Serotonin

The API I selected is 'Dog API'. This API responds with a random image of a dog and can even filter by breed and generate lists of images by breed. It can be found here Dog API.


Dog API's response includes two headers: a message aand a status. I retrieved the message, which includes the randomly generated image of the dog. Dog API's message header in this case holds only one image, but in other cases (filtering by breed), it can hold an array of random images of dogs of that breed.
